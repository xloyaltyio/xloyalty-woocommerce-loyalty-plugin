Description
The Xloyalty.io Basic Plugin integrates a loyalty points system into your WooCommerce store. Customers earn points for every purchase, which they can redeem for discounts. The plugin includes settings to customize points allocation and the redemption rate.

When you activate the plugin, you will be prompted to fill out a short form so we can learn more about you. By activating the plugin, you agree to our Privacy Policy. We will contact you only for the following purposes:

To conduct a satisfaction survey regarding the free Xloyalty.io plugin.
To provide free support via email.
To present the features of the Xloyalty.io Premium Plugin.
There are no financial or other obligations for activating the free version of the plugin for your eShop. For free support, please contact us at support@xloyalty.io. We will respond within 5 business days.

Installation
Download the plugin's .zip file.
Upload the plugin via WordPress Admin → Plugins → Add New → Upload Plugin.
Activate the plugin through the Plugins menu in WordPress.
Configure the plugin under WooCommerce → Settings → Xloyalty.io.
Features
Earn points for each completed WooCommerce order.
Customize points earned per Euro (x) and redemption rate (y).
Display points in the cart and on the customer account page.
Redeem points directly at checkout.
Settings page integrated with WooCommerce.
